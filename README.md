# Rain World - Rotund World Demo

A visual recreation of the "Rotund World" mod for Rain World, where creatures become comically bloated and can barely move.

## Features

- **Bloated Creatures**: Oversized, wobbling creatures that struggle to move
- **Physics**: Gravity affects movement, creatures bounce off walls
- **Arena**: An orange-bordered arena with a shelter area
- **Rain Particles**: Orange rain drops falling from above
- **Interactive**: Click "Clear" to respawn creatures

## How to Run

1. Open `index.html` in a modern web browser
2. Watch the rotund creatures struggle to move!
3. Click the "Clear" button to respawn them

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling for the UI and canvas
- `script.js` - Game logic and rendering

## Gameplay

The demo shows various creatures affected by the "Rotund World" effect:
- Green creatures are extremely bloated (rotund)
- Orange/red creatures are normal sized for comparison
- All creatures fall towards the shelter area at the bottom
- Creatures wobble due to their bloated state
